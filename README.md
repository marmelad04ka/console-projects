# Number Guessing Game

A console-based number guessing game where players try to guess a randomly generated number between 1 and 100 with progressive hints.

## Features

- **Three difficulty levels:**
  - Easy (10 chances)
  - Medium (5 chances)
  - Hard (3 chances)
- **Progressive hint system** (optional):
  - Hint 1: Which half the number is in
  - Hint 2: Which quarter
  - Hint 3: Which ten
  - Hint 4: Three possible numbers including the answer
- **Leaderboard** tracking best scores for each difficulty

## Requirements

- Python 3.6 or higher
- No external dependencies (uses only standard library)

## How to Run

1. **Clone the repository:**
   ```bash
   git clone https://github.com/marmelad04ka/console-projects.git
   cd console-projects
   ```
2. **Run the game:**
  ```bash
   python number_guessing_game.py
  ```
  or
  ```bash
   py number_guessing_game.py
  ```
**If you're using Python 3 specifically:**
  ```bash
   python3 number_guessing_game.py
  ```
